# BF4 Server Notifier

A simple Chrome extension that monitors **Battlefield 4 servers** and notifies you when servers match your custom filters (maps, game modes, presets, and minimum player count).

## 🔧 Features

- Auto-refresh every 60 seconds.
- Filter by:
  - Maps (multiple DLCs supported)
  - Game modes (Conquest, Rush, TDM, etc.)
  - Server presets (Normal, Hardcore, Custom, etc.)
  - Minimum number of players
- Desktop notifications when a matching server is found.
- Toggle the extension on/off.
- Clean and simple UI.

## 📥 Installation (Manual)

1. Open Google Chrome and go to:

2. Enable **Developer mode** (top-right switch).

3. Click **"Load unpacked"**.

4. Select the folder where you downloaded/cloned this extension.

5. The extension should now appear in your list and be ready to configure and use!

## ⚙️ How to Use

1. Go to the official server browser:
https://battlelog.battlefield.com/bf4/servers/

2. Click the extension icon and set your preferences:
- Choose the maps you want.
- Optionally set game modes and presets.
- Set the minimum number of players you want.
- Make sure at least one map is selected.

3. Save your preferences.

4. The extension will automatically reload the page every 60 seconds and notify you when a matching server appears.

## 🧪 Notes

- You must stay on the **Battlelog servers page** for the extension to work.
- You must pre-filter with the website filters, then the extension will track in the informations in the screen.
- The extension only works with the **Battlelog** version of BF4 (not the in-game browser).
